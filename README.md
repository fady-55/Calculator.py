# Python Calculator

A simple calculator written in Python as a personal project after completing Harvard CS50.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Exponentiation
- Save calculation history
- Error handling

## Project Structure

```
calculator.py
operations.py
history.py
history.txt
```

## Run

```bash
python calculator.py
```

## Example

```
Choose: 1
First number: 20
Second number: 15

Result = 35
```
