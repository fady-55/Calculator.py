# 🧮 CS50 Python Calculator

A command-line calculator developed in Python as a personal project after studying Harvard's CS50.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Power
- Modulus
- Square Root
- Operation History
- Clear History
- Error Handling
- Clean Code Structure

---

## Requirements

- Python 3.10+

---

## Run the Program

```bash
python calculator.py
```
